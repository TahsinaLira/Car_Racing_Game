 Car Racing Game (Java)
 
This is a simple car racing game developed in Java where the player controls a car and avoids collision with incoming bot cars. The goal is to survive as long as possible and achieve a high score.

Technologies Used
•	Java (Swing for GUI)
•	JDBC 
•	MySQL 
•	Multithreading 


How to Run
1.	Clone the repository:
2.	git clone https://github.com/TahsinaLira/Car_Racing_Game.git
4.	Open the project in any Java IDE (IntelliJ / Eclipse / NetBeans)
5.	Set up MySQL database:
o	Create a database 
o	Import the SQL file (if provided)
6.	Update database credentials in the code:
7.	String url = "jdbc:mysql://localhost:3306/database_name";
8.	String user = "root";
9.	String password = "your_password";
10.	Run the main file: Main.java
